# Live Location App

A real-time location sharing application that allows users to share their current location with others, similar to WhatsApp's live location feature.

## Features

- **Share Your Location**: Broadcast your real-time location
- **Track Friends**: View the live location of users who've shared with you
- **Secure Sharing**: Control who can access your location and for how long
- **Real-time Updates**: Location updates reflected instantly across all connected devices
- **Location History**: Optional tracking of shared location sessions
